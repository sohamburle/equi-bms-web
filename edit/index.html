Instructions -


For adding new entry:

  1. Fill up the entire form from the respective section that you want to add to.

For submitting a correction or updating info:

  1. Start typing in the Name column, autocomplete will complete the name for you!
    Incase if the name doesn't autocomplete that means the record isn't available in my repository, please select add instead of edit!
    (Please wait for a few seconds for the autocomplete to respond!)
  2. Fill up all the important fields in that form.
  3. Only fill up those fields that you want to change. Rest all can be left blank.
