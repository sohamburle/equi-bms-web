Changelog



Version 0.10

  1. Added and completed Merch section.
  2. Reworked on how data is downloaded. Data download should be quick now. (Depends on your net speed though. Please be patient.)
    PS. Versions prior to this will still display data but will not download it as fast as the this version does.
  3. Added a splash screen for a duration of 2 seconds. (Splash screen image is made by Chinmay Salaskar!)
  4. Removed subsections of Events section.
  5. Custom main layout now looks perfect on almost all devices.
  6. Replaced 'Connection not available' toast message with a 'no connection' layout.
  7. Added list item click animation.
  8. Removed update check on every app launch. Please check for update manually from main menu.
  9. Strengthened the refresh data logic. Refresh data is only available under main menu.

Version 0.9

  1. Completed Venues section.
  2. Added a button in Jam Rooms' detail screen, on clicking it the location of that jam room will be shown on Google Maps.
  3. Merged Reviews and News, now it's under News.
  4. Removed Managers | Organizers because of lack of data and necessity.
  5. Added the current version name of app in the main screen menu.
  6. Few layout improvements.

Version 0.8

  1. Implemented a custom layout replacing Navigation Drawer. (Images used in the layout are made by Chinmay Salaskar!)
  2. Merged Jam Rooms and Recording Studios, now it's under Music Studios.
  3. Added a button to show Album tracks in Artists' detail screen. (button will be visible only if the Artist has released something.)
  4. Added contact details to Artists' detail screen.
  5. Added an option to 'check for update' in menu of the main screen.
  6. Made the app force Portrait layout.
  7. Few other implementations.

Version 0.7

  1. Reworked on how images are displayed in the list.
  2. Fixed a bug in Jam Rooms detail view (introduced in previous version).
  3. Added Launcher Icon. (Icon made by Chinmay Salaskar!)
  4. Implemented Material Design for Update and Changelog Dialog.
